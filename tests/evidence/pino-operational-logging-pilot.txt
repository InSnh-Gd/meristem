Pino operational logging pilot — m-eventbus + m-log
====================================================

Date: 2026-06-19
Scope: Replace console.log/warn/error with pino structured logging in m-eventbus
and m-log. Operational logging only — M-Log authoritative Timeline/Full/Audit
fact emission (DB writes via write-service + OpenSearch projection) is untouched.

Factory
-------
File: packages/telemetry/src/logger.ts
Export: createLogger(serviceName: string) => pino.Logger
- Level via MERISTEM_LOG_LEVEL env var (default 'info').
- Production (NODE_ENV=production): JSON lines to stdout.
- Non-production: self-contained minimal pretty-printer (no pino-pretty dep).
- Exposed via packages/telemetry package.json "./logger" subpath and re-exported
  from packages/telemetry/src/index.ts barrel.

Replaced console.* sites
------------------------
m-eventbus/src/index.ts:
  - console.log("m-eventbus listening ...")  -> logger.info({url}, 'm-eventbus listening')
m-eventbus/src/publisher.ts:
  - console.error("[m-eventbus] dlq_publish_failed", {...})  -> logger.error({...}, 'dlq_publish_failed')
m-log/src/index.ts:
  - console.warn("m-log: OpenSearch unavailable ...")  -> logger.warn('opensearch unavailable ...')
  - console.warn("m-log: projection engine unavailable ...")  -> logger.warn('projection engine unavailable ...')
  - console.warn("m-log: ${dependency} readiness probe degraded ...")  -> logger.warn({dependency, error}, 'readiness probe degraded')
  - console.log("m-log listening ...")  -> logger.info({url}, 'm-log listening')
m-log/src/eventbus-operational-consumer.ts:
  - console.warn("[m-log] eventbus_operational_consume_failed", {...})  -> logger.warn({...}, 'eventbus_operational_consume_failed')
m-log/src/write-service.ts:
  - console.warn("m-log: failed to index ... into OpenSearch - ...")  -> logger.warn({kind, entryId, error}, 'opensearch_index_failed')
m-log/src/projection/health.ts:
  - console.warn("m-log projection: failed to probe OpenSearch health - ...")  -> logger.warn({error}, 'opensearch_health_probe_failed')
m-log/src/opensearch.ts:
  - console.warn("m-log: OpenSearch ${operation} degraded - ...")  -> logger.warn({operation, error}, 'opensearch_degraded')
