# M-UI Design Tool Availability

> Historical reference only. This file records which design tools were available during an earlier M-UI exploration pass. It does not define current frontend direction.

## Preserved Notes

- Stitch skills can help structure prompts and concepts, but they require a configured backend before any generation or upload workflow is real.
- Figma validation requires an accessible Figma file and MCP credentials before read-only inspection can happen.
- Design-system extraction tools are useful for documenting an already-converged frontend, not for forcing a design direction during this reboot.

## Current Rule

Use the current task requirements first. Treat design tools as optional accelerators only after the desired UI direction has been restated for the new work.
